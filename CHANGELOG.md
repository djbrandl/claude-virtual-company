# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-31

### Added
- Initial release of Claude Virtual Company framework
- Core role skills: CTO, Architect, Tech Lead, Senior Dev, Developer, QA
- Hiring Manager for dynamic specialist creation
- Git Flow expertise with branch strategy and PR workflows
- Code Reviewer specialist for quality gates
- Test Architect for comprehensive testing strategy
- Proposal-based governance system
- Task dependency tracking with sync protocol
- Configurable company settings
- Slash commands: `/company`, `/company-settings`, `/company-merge`, `/company-status`, `/company-roster`, `/company-hire`
- File-based inbox system for agent communication
- Quality gates with mandatory testing
- Support for Puppeteer/Playwright UI testing
- Comprehensive documentation

### Security
- Governance matrix for role-based permissions
- Proposal approval workflow for cross-role task changes
- CEO escalation for critical decisions
